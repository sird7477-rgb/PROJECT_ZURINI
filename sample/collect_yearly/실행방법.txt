대신증권 CYBOS 수집기 실행 방법

전제 조건
- Windows 32비트 Python 환경
- CYBOS Plus 로그인 완료
- 관리자 권한 프롬프트
- pywin32 설치

예시

set CONDA_FORCE_32BIT=1
conda activate 32bit_collector

D:
cd D:\gazuaaaaa\collect_yearly

python collect_yearly.py --output-dir D:\gazuaaaaa\collect_yearly\data --months 24

명시 기간 수집

python collect_yearly.py --output-dir D:\gazuaaaaa\collect_yearly\data --start-date 20250101 --end-date 20251231

종목 1분봉만 수집

python collect_yearly.py --no-indices --no-metadata

지수 1분봉만 수집

python collect_yearly.py --no-stocks --no-metadata

출력 구조

data/
  minute-bars/YYYYMM/A000020.csv
  index-bars/YYYYMM/U001.csv
  symbols/symbols_YYYYMMDD_HHMMSS.csv
  symbols/indices_YYYYMMDD_HHMMSS.csv
  manifests/collection_manifest_YYYYMMDD_HHMMSS.jsonl

주의
- 이 스크립트는 실제 계정 비밀번호나 인증서 암호를 저장하지 않는다.
- 중단 후 재실행하면 이미 존재하는 CSV는 건너뛴다.
- 2차 백테스트용 기본 지수 코드는 U001=KOSPI, U201=KOSDAQ, U180=KOSPI200이다.
